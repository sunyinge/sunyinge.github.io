Set up

  1. Create a repo named username.github.io, where username is your username on GitHub. If you have already uploaded to other repo, rename the repo instead.

  2. Follow this page to install Hexo and deploy your new blog.

  3. In order to show mathematical formulas on blog pages, you can choose MathJax to render them and use hexo-renderer-pandoc as Hexo's Markdown renderer. To install hexo-renderer-pandoc, follow the instructions here if you are using the theme next and see the post for potential problems you might encouter. If you are using paperbox as your hexo theme, run the following scripts in your terminal under the path ~/***/username.github.io.

  4. If you need to post an image, create an images foler in source (_posts does not work for me) folder and follow instructions here hexo-asset-link and Root-relative link.

    You're all set.
