Measure of accuracy

1. F1 Score

2. Multiclass Classification

Based on the confusion matrix (left), we can calculate the precision and recall for each class (right).

For the following first two average, calculate F1 score per class first.

  • Macro-averaged
    • Treat each class equally
    • Mean of per-class F1-scores
    • Macro-averaged precision and macro-averaged recall are calculated in the similar way
  • Weighted-averaged
    • Weight the F1-score of each class by the number of samples from that class
    • Weighted-averaged precision and weighted-averaged recall are calculated in the similar way
  • Micro-averaged
    • Look at all the samples together
    • Each prediction error (X is misclassified as Y) is a False Positive for Y, and a False Negative for X so FP = FN and therefore true = TP+FN = TP+FP = positive.
    • micro-F1 = micro-precision = micro-recall

When there is class imbalance, micro average is preferable.

3. Multilabel classification

  • Label based

    • same as multiclass classification, i.e., macro, weighted and micro.

  • Sample based

    • \(n\) is the number of examples.
    • \(Y_i\) is the ground truth label assignment of the \(i\)th example..
    • \(x_i\) is the \(ùëñ\)th example.
    • \(h(x_i)\) is the predicted labels for the \(i\)th example.

    Then,

    • Precision = \(\frac{1}{n}\sum_{i=1}^n\frac{|Y_i\cap h(x_i)| }{|h(x_i)|}\), the ratio of how much of the predicted is correct.
    • Recall = \(\frac{1}{n}\sum_{i=1}^n\frac{|Y_i\cap h(x_i)| }{|Y_i|}\), the ratio of how many of the actual labels were predicted.
    • F1 score = harmonic mean of precision and recall

2. Receiver Operating Characteristic (ROC)

Pairs of true positive rates and false positive rates at different thresholds.

  • True positive rate = recall = \(\frac{true \ positives}{true}\)
  • False positive rate = \(\frac{false \ positives}{false}\)
    • 1- specificity
