Transfer learning

  • Layer Transfer
    • Fine-tune the model (all model layers are updated)
    • Keep the copied layers from large models unchanged
    • Remark: speech copy the last few layers while image copy the first few layers
  • Conservative Training
    • Old model: trained on source data
    • New model: trained on new/targeted data
    • Add constraints so that the outputs on same input data from the old model and the new model are similar; or L2 norm of parameter differences of the old model and the new model is small.
